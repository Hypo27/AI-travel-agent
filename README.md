# AI Travel Planning Agent ✈️


This is an AI-powered travel agent built for the Google Developers Club task. It uses Google Gemini and Tavily Search to create personalized travel itineraries based on user interests and budget.


- Smart Itinerary Planning: Generates day-by-day plans.
- Budget Calculator: Estimates costs using a custom tool.
- Real-time Search: Fetches up-to-date hotel and flight info.


1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
